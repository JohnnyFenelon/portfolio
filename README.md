## 🌟 About the Project

This is a modern, high-performance personal portfolio designed with a **"Noir Gold Futurism"** aesthetic. It is built to be fully responsive, highly interactive, and trilingual, reflecting my ability to bridge the gap between executive leadership, engineering, and global communication.

### ✨ Key Features
- 🌐 **Trilingual Support:** Seamless switching between Spanish, English, and French.
- 📱 **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
- 📧 **Functional Contact Form:** Integrated with FormSubmit API for reliable email delivery without server-side mail configuration.
- 📰 **Newsletter Subscription:** Custom PHP backend connected to a MySQL database to capture and manage subscriber leads.
- 🎨 **Modern UI/UX:** Custom CSS variables, smooth scroll animations, and an interactive HTML5 Canvas particle background.
- 🖼️ **Multimedia Gallery:** Interactive photo and video sections with a custom-built lightbox.
- 🔐 **Authentication Ready:** Frontend structure prepared for Supabase integration (Login/Register).

---

## 🛠️ Tech Stack

### Frontend
- **HTML5** (Semantic markup)
- **CSS3** (Custom Properties/Variables, Flexbox, Grid, Animations)
- **Vanilla JavaScript** (ES6+, Fetch API, DOM Manipulation, Intersection Observer)
- **Supabase JS SDK** (Prepared for Auth/Database integration)

### Backend & Services
- **PHP** (Handling newsletter subscriptions and database connections)
- **MySQL** (Storing subscriber data securely)
- **FormSubmit.co** (AJAX-based contact form routing)

 Local Installation & Setup
If you want to run this project locally or contribute to it, follow these steps:
Prerequisites
A local server environment like XAMPP, WAMP, or MAMP (required for PHP execution).
A modern web browser (Chrome, Firefox, Edge).
Steps
Clone or Download this repository to your local machine.
Move the project folder into your server's root directory (e.g., C:\xampp\htdocs\academia809).
Start Apache (and MySQL if you want the subscription form to work) from your XAMPP Control Panel.
Open your browser and navigate to: http://localhost/academia809
Database Setup (For Newsletter Feature)
If you want the subscription form to work locally:
Open phpMyAdmin (http://localhost/phpmyadmin).
Create a new database named academia_subs.
Import or run the following SQL to create the table:
sql
123456
Update the database credentials in subscribe.php to match your local XAMPP settings (usually root with no password).
🚀 Deployment
This portfolio is currently hosted on Namecheap Shared Hosting using cPanel.
Deployment Steps:
Log in to Namecheap cPanel and open File Manager.
Navigate to the public_html directory (or the specific addon domain folder).
Upload index.html, subscribe.php, and PROFILE_1.png.
Configure the MySQL database via MySQL Databases in cPanel and update the credentials in subscribe.php.
The site is instantly live!
